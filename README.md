# POO_Python
Exercícios de POO em Python.

Neste projeto, você deverá desenvolver um sistema bancário simplificado utilizando os principais conceitos de Programação Orientada a Objetos (POO) em Python, como:

-Herança
-Abstração
-Encapsulamento
-Polimorfismo

Classes Principais
-Cliente
-Conta
-ContaCorrente
-ContaPoupanca
-Transação
